# StatusBarHud
一个简单的状态栏指示器



### 显示提示信息

```objc
[StatusBarHUD showMessage:@"提示信息"];
```

### 显示成功信息

```objc
[StatusBarHUD showSuccess:@"加载成功"];
```

### 显示失败信息

```objc
[StatusBarHUD showError:@"加载失败"];
```

### 显示加载信息

```objc
[StatusBarHUD showLoading:@"正在加载中"];
```

### 显示隐藏信息

```objc
[StatusBarHUD hide];
```
